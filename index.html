Las Partes del Computador

Antes de comenzar, repasemos las partes más importantes del computador:

  • CPU (Unidad Central de Procesamiento): Es el cerebro del computador, donde se ejecutan los programas.
  • Monitor: Muestra la información visual al usuario.
  • Teclado: Permite ingresar texto y comandos.
  • Ratón o Mouse: Dispositivo apuntador para interactuar con la interfaz gráfica.
  • Memoria RAM: Almacena temporalmente los datos que el CPU necesita procesar rápidamente.
  • Disco Duro: Guarda los archivos y programas de forma permanente.
  • Tarjeta Madre: Conecta y comunica todos los componentes del sistema.
  • Fuente de Poder: Suministra la energía eléctrica al computador.
