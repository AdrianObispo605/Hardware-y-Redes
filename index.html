¡Bienvenido a la Actividad sobre Partes del Computador!

Las principales partes de un computador incluyen:

  • CPU (Unidad Central de Procesamiento): El "cerebro" que ejecuta instrucciones.
  • Monitor: La pantalla que muestra la información visual.
  • Teclado: Dispositivo para ingresar texto y comandos.
  • Ratón: Herramienta para navegar y seleccionar en la interfaz gráfica.
  • Memoria RAM: Almacenamiento temporal para datos en uso rápido.
  • Disco Duro o SSD: Almacenamiento permanente para archivos y programas.
  • Placa Madre: Conecta todos los componentes.
  • Tarjeta Gráfica: Maneja el procesamiento de imágenes.
